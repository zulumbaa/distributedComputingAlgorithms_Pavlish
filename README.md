# Distributed Computing Algorithms

This repository contains a study project for the university course **Distributed Computing Algorithms**.

The goal of this project is to implement and experiment with classic distributed algorithms and understand how processes communicate and coordinate in a distributed system.

The project will include small simulations and implementations of different algorithms used in distributed computing.

## Technologies

* Programming language: * Python*
* Message passing simulation between processes

## Implemented Tasks

### Task 1 – Logical Clocks

Implementation of:

* **Lamport Logical Clock**
* **Vector Clock**

Both clocks are implemented based on the **Tarry algorithm simulation**.

The implementation demonstrates how distributed processes maintain a logical ordering of events without a global clock.

Reference implementation idea:
https://github.com/NickRenton/dss/tree/tarry

## Purpose

This repository is created for educational purposes as part of the university coursework.
